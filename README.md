## 🚀 Installation & Setup
### 1. Clone the repository
```
git clone https://github.com/Mahmoud-Saadaoui/facebook-clone.git
cd facebook-clone
```
### 2. Install dependencies
```
npm install
```
### 3. Run the development server
```
npm start
```
